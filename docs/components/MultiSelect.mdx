# MultiSelect

Component for selecting multiple options from a list of options.
Supports clearable, controlled, and customizable behavior.

import {
  BlossomUIRenderer,
  PropsTable,
  MultiSelectUsage,
  MultiSelectDefaultValue,
  MultiSelectMode,
  MultiSelectClearable,
  MultiSelectControlled,
  MultiSelectDisabled,
  MultiSelectCustomStyle,
  MultiSelectCustomPickerStyle,
  MultiSelectMaxSelect,
  MultiSelectObjectOptions,
} from "@site/src/components";

The following option data will be used in the below demos:

```tsx
const OPTIONS = Array(25)
  .fill(0)
  .map((_, i) => ({
    label: `Option ${i}`,
    value: i,
    disabled: i % 4 === 0,
  }));
```
