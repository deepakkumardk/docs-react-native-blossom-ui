# ExpandableText

Component to display long text that can be expanded or collapsed.  
Supports controlling number of visible lines, custom labels, and line spacing.

import {
  BlossomUIRenderer,
  PropsTable,
  ExpandableTextUsage,
  ExpandableTextLines,
  ExpandableTextLessLines,
  ExpandableTextLabelCustom,
} from "@site/src/components";

## Usage

<BlossomUIRenderer
  componentName={"ExpandableTextUsage"}
  children={<ExpandableTextUsage />}
  fileName={"ExpandableText"}
/>

## Less Lines

Use the `numberOfLines` prop to limit how many lines are visible before expansion.

<BlossomUIRenderer
  componentName={"ExpandableTextLessLines"}
  children={<ExpandableTextLessLines />}
  fileName={"ExpandableText"}
/>

## Lines

Again, use the same `numberOfLines` prop to limit how many lines are visible before expansion.

<BlossomUIRenderer
  componentName={"ExpandableTextLines"}
  children={<ExpandableTextLines />}
  fileName={"ExpandableText"}
/>

## Label

Use `showLabel` and `hideLabel` props to customize the toggle text & their corresponding styles.
