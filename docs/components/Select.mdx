# Select

Single select picker component for primitive values or objects.  
Supports default values, clearable selection, custom items, and full styling.

import {
  BlossomUIRenderer,
  PropsTable,
  SelectUsage,
  SelectDefaultValue,
  SelectClearable,
  SelectCustomItem,
  SelectCustomStyle,
  SelectCustomPickerStyle,
  SelectDisabled,
  SelectObjectOptions,
} from "@site/src/components";

The following option data will be used in the below demos:

```tsx
const OPTIONS = Array(25)
  .fill(0)
  .map((_, i) => ({
    label: `Option ${i}`,
    value: i,
    disabled: i % 4 === 0,
  }));
```

## Usage

<BlossomUIRenderer
  componentName={"SelectUsage"}
  children={<SelectUsage />}
  fileName={"Select"}
/>

## Default Value

Use `defaultValue` to preselect an item when the component mounts.

<BlossomUIRenderer
  componentName={"SelectDefaultValue"}

  fileName={"Select"}

## Clearable

Enable the `clearable` prop to allow users to remove all selected options at once from the right clear icon.

<BlossomUIRenderer
  componentName={"SelectClearable"}

/>
