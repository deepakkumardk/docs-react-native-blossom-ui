# Spacer

Component to add spacing between elements.  
Supports horizontal and vertical spacing, with optional background color for visibility.

import {
  BlossomUIRenderer,
  PropsTable,
  SpacerUsage,
  SpacerVertical,
  SpacerOnBackground,
} from "@site/src/components";

## Usage

<BlossomUIRenderer
  componentName={"SpacerUsage"}
  children={<SpacerUsage />}
  fileName={"Spacer"}
/>

## Vertical

Use the `width` & `height` prop in conjunction to have the vertical look.

<BlossomUIRenderer
  componentName={"SpacerVertical"}
  children={<SpacerVertical />}
  fileName={"Spacer"}
/>

## On Background

Space has the transparent background so it does takes the parent background color.

<BlossomUIRenderer
  componentName={"SpacerOnBackground"}
  children={<SpacerOnBackground />}
  fileName={"Spacer"}
/>
