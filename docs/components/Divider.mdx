# Divider

Component to separate content visually with horizontal or vertical lines.  
Supports labels, custom spacing, and styling for better layout structure.

import {
  BlossomUIRenderer,
  PropsTable,
  DividerUsage,
  DividerLabel,
  DividerSpacing,
  DividerVertical,
  DividerCustom,
} from "@site/src/components";

## Usage

<BlossomUIRenderer
  componentName={"DividerUsage"}
  children={<DividerUsage />}
  fileName={"Divider"}
/>

## Vertical

Use width & height prop in conjunction to have the vertical look.

<BlossomUIRenderer
  componentName={"DividerVertical"}
  children={<DividerVertical />}
  fileName={"Divider"}
/>

## Spacing

use the `spacing` prop to have the around cross axis i.e vertical margin for horizontal line and horizontal margin for vertical line.

<BlossomUIRenderer
  componentName={"DividerSpacing"}
  children={<DividerSpacing />}
  fileName={"Divider"}
/>

## Label

Use the `label` & `labelPosition` prop together to show and control the position of label (`left`, `center` & `right`)

<BlossomUIRenderer
  componentName={"DividerLabel"}
  children={<DividerLabel />}
  fileName={"Divider"}
/>
