# Typography

Typography supports below font sizes with corresponding weights. User have the option to override it's behavior and make corresponding changes.

| name      | typography | fontSize | fontWeight | comment                     |
| --------- | ---------- | -------- | ---------- | --------------------------- |
| heading1  | **h1**     | 36       | 700        |                             |
| heading2  | **h2**     | 32       | 700        |                             |
| heading3  | **h3**     | 28       | 700        |                             |
| heading4  | **h4**     | 24       | 600        |                             |
| heading5  | **h5**     | 22       | 600        |                             |
| heading6  | **h6**     | 20       | 600        | Default for Header Title    |
| subtitle1 | **s1**     | 18       | 600        |                             |
| subtitle2 | **s2**     | 18       | 500        |                             |
| subtitle3 | **s3**     | 18       | 400        |                             |
| body1     | **b1**     | 16       | 600        | Default for buttons         |
| body2     | **b2**     | 16       | 500        | Default for Text            |
| body3     | **b3**     | 16       | 400        |                             |
| label1    | **l1**     | 14       | 600        |                             |
| label2    | **l2**     | 14       | 500        | Default for Labels          |
| label3    | **l3**     | 14       | 400        | Default for Labels          |
| caption1  | **c1**     | 13       | 400        | Default for Captions/errors |
| caption2  | **c2**     | 12       | 500        |                             |
| caption3  | **c3**     | 12       | 400        |                             |
